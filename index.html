/** * 日本語 → ベトナム語 翻訳API * GASウェブアプリ * * 通常JSON / JSONP 両対応版 * HTMLをGitHub Pages等の別ドメインで開く場合はJSONP対応が安全です。 */ function doGet(e) { return handleTranslate(e); } function doPost(e) { return handleTranslate(e); } function handleTranslate(e) { var out = { translated: "", error: null }; try { var text = ""; if (e && e.parameter && e.parameter.text) { text = e.parameter.text; } if (!text && e && e.postData && e.postData.contents) { try { var body = JSON.parse(e.postData.contents); text = body.text || ""; } catch (jsonErr) { text = e.postData.contents || ""; } } text = String(text || "").trim(); if (!text) { out.error = "翻訳する文字がありません"; } else { out.translated = LanguageApp.translate(text, "ja", "vi"); } } catch (err) { out.error = String(err); } var json = JSON.stringify(out); // JSONP対応: ?callback=関数名 がある場合は JavaScript として返す var callback = e && e.parameter && e.parameter.callback; if (callback) { callback = String(callback).replace(/[^\w.$]/g, ""); return ContentService .createTextOutput(callback + "(" + json + ");") .setMimeType(ContentService.MimeType.JAVASCRIPT); } // 通常JSON return ContentService .createTextOutput(json) .setMimeType(ContentService.MimeType.JSON); }
